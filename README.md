# Gemynd Brand & Strategy 2026

**Public working repository** for Gemynd's brand system, marketing assets, and strategic planning.

## Purpose

This repo holds the foundational materials for Gemynd's dual-identity positioning:

- **WestFlow** — The Cognitive Orchestration Layer (platform + AI Swizzle methodology)
- **Gemynd** — The Grounded Memory Platform (human legacy & memory vertical, powered by WestFlow)
