## The Strategic Reality We Are Solving

Gemynd operates at the intersection of two powerful but tonally different worlds:

### 1. WestFlow (The Platform)
- **Positioning:** "The Cognitive Orchestration Layer"
- **Audience:** Copier dealers, SMB operations, document-heavy businesses, TWAIN partners, field service, HR teams
- **Tone:** Capable, technical, practical, fast, grounded in documents and workflows
- **Products:** Aiva (workforce), Katie (field service), FlowView (orchestration), general agent builder
- **Strength:** AI Swizzle methodology — raw documents/scans → reliable Digital Workers in hours

### 2. Gemynd / StoryScribe (The Human Legacy Layer)
- **Positioning:** "The Grounded Memory Platform"
- **Audience:** Funeral professionals, senior living, families, genealogists, pet owners, corporate culture teams
- **Tone:** Dignified, warm, premium, emotionally resonant, trustworthy
- **Flagship:** StoryScribe — cinematic legacies and structured memory reports with explicit provenance
- **Beachhead:** Death care via NFDA October 2026 (Epson + Passare)

**The Shared Truth:** Both use the exact same core technology (multimodal grounded extraction + agentic narrative engine + provenance layer).

**The Opportunity:** We do not need to choose. We need a brand architecture that lets each side speak in its authentic voice while clearly showing they come from the same powerful engine.

---

## Recommended Brand Architecture

**WestFlow** = The Engine & Platform  
**Gemynd** = The Memory & Legacy Application Layer (powered by WestFlow)

### Core Narrative Line (Internal & External)

> "WestFlow turns documents and conversations into reliable, grounded agents that get work done.  
> Gemynd applies that same provenance-first engine to the stories and memories that define who we are."

This single line protects both sides.

### Visual & Verbal Separation (With Connection)

| Element              | WestFlow                              | Gemynd (StoryScribe)                     | Connection Signal          |
|----------------------|---------------------------------------|------------------------------------------|----------------------------|
| Primary Color        | Deep technical navy + teal accents    | Deep heritage navy + warm gold           | Shared deep navy           |
| Secondary            | Clean, modern sans                    | Refined sans + subtle heritage warmth    | Shared typography family   |
| Photography          | Clean workspaces, scanning hardware, professionals in action | Families, seniors, meaningful objects, archival textures, cinematic stills | Both feel "grounded"       |
| Voice                | Direct, capable, "replace search bars with workers" | Warm, dignified, "every story grounded in truth" | Both emphasize provenance  |
| Logo Treatment       | Bold, modern wordmark                 | Elegant wordmark + refined icon          | Shared "G" monogram option |
