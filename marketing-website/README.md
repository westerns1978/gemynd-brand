# Gemynd Marketing Website

Premium, emotionally resonant marketing site for the Gemynd Grounded Memory Platform (StoryScribe flagship).

## How to View

1. Clone or download this repository.
2. Open `marketing-website/index.html` directly in any modern browser.

The site is fully self-contained (Tailwind via CDN + inline styles). No build step required.

## Purpose

This is the high-end, trust-focused experience designed for:
- Funeral professionals (NFDA 2026 beachhead)
- Senior living / post-acute facilities
- Families (DTC legacy stories)
- Genealogists and pet owners

It emphasizes provenance, emotional resonance, and the "every story grounded in truth" positioning.

## Relationship to WestFlow

This site represents the **Gemynd** face of the platform. It can live at gemynd.gemyndflow.com (or storyscribe.gemyndflow.com) while gemyndflow.com focuses on the broader WestFlow platform story.
